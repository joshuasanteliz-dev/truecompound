# Design Reference Extractor

Analyze visual references without copying them.

For each reference, extract:
- typography direction
- color logic
- layout rhythm
- spacing density
- hierarchy patterns
- imagery/art direction
- interaction/motion style
- brand personality
- psychological effect
- why it feels premium, useful, memorable, or trustworthy
- what would be inappropriate to copy directly

Then synthesize:
1. Shared patterns across references.
2. Contradictions between references.
3. Which patterns fit the current project.
4. Which patterns do not fit the current project.
5. A unique design direction inspired by the references but not derivative.

Rules:
- Do not clone.
- Do not copy brand assets.
- Do not copy distinctive layouts one-to-one.
- Extract principles, not pixels.
- Adapt the design language to the actual audience and goal.
